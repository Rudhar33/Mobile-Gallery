Device Description

The Google Pixel 7. The successor to the Pixel 6. Now with new exciting features.

The smartphone features the Google Tenosr G2 chip which is a big improvement from the Tensor G1. The chipset is same as the Pixel 7 pro. The smartphone has gorilla glass on both sides. At the back it has gorilla glass victus which ennsures that the phone could handle some drops. The display is 6.3 inch OLED with 90 Hz refresh rate, has HDR 10+ and deep blacks. The device has a in display fingerprint sensor which is best when it comes to security. There are two storage options 128 or 256GB which is non expandable. The main camera is 50MP with a 12MP ultrawide camera. The rear camera is also capable of recording 4k videos with excellent dynamic range.
